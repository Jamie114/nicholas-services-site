# Loan Web App V3.2

This revision adds styled branded PDF exports for each tab using jsPDF.

## Update steps
1. Replace `src/App.tsx`, `src/calculations.ts`, `src/styles.css`, and `package.json`.
2. In your project folder run `npm install`.
3. Start again with `npm run dev`.
